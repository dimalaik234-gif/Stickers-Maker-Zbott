# Telegram Sticker Pack Creator Bot

A production-ready Telegram bot built with aiogram 3.x that converts user images into custom sticker packs with background removal capabilities.

## Features

- 🎨 Automatic background removal using rembg
- 🖼 Option to keep original background
- 🌐 Multi-language support (Russian/English)
- 💾 Personal sticker pack management per user
- ⚡ Fully asynchronous and memory-efficient
- 🔒 SQLite database for user preferences
